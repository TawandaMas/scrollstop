# StressBall PWA

A progressive web app for stress relief with three interactive exercises.

## Features

1. **Infinite Bubble Wrap**
   - Tap to pop bubbles with satisfying haptic feedback
   - Auto-resets when all bubbles are popped
   - Vibration support on mobile devices

2. **Kaleidoscope Drawing**
   - Create beautiful symmetric patterns
   - Change colors and segment counts
   - Touch and mouse support

3. **Focus Labyrinth**
   - Tilt device or use touch to guide ball
   - Randomly generated mazes
   - Gyroscope support

## PWA Features

- ✅ Works offline
- ✅ Installable on mobile devices
- ✅ Fast loading
- ✅ Responsive design
- ✅ Dark/Light theme

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this directory
3. Go to repository Settings → Pages
4. Select "main" branch as source
5. Your app will be available at: `https://[your-username].github.io/[repository-name]`

## Local Development

Just open `index.html` in a browser. No build process required!

## Technologies Used
